 Rotina da Constante de Kaprekar (6174)

Este repositório contém a atividade avaliativa de lógica de programação para a disciplina de Raciocínio Lógico e Algoritmos. O objetivo é demonstrar o processo matemático que leva qualquer número de quatro dígitos (com no máximo dois repetidos) à Constante de Kaprekar.

 Objetivo
Desenvolver um algoritmo que execute a Rotina de Kaprekar, exibindo o passo a passo das iterações até atingir o valor 6174.

  Restrições Técnicas Aplicadas
  
Conforme as exigências do edital, o código foi desenvolvido **sem** a utilização de:
 Bibliotecas externas.
 
  Funções prontas de ordenação.
  
Arrays ou Listas.

Processamento de texto (Strings) para manipulação de dígitos.

Toda a lógica de separação, ordenação (NDC/NDD) e validação foi implementada exclusivamente através de **operações aritméticas** (divisão inteira e resto), estruturas de repetição (`while`, `for`) e condicionais (`if`, `else`).

 Requisitos do Algoritmo
1. Validação de tipo e faixa (0000 a 9999).
2. Validação de repetição de dígitos (máximo 2 iguais).
3. Construção do NDC (Dígitos Crescentes).
4. Construção do NDD (Dígitos Decrescentes).
5. Subtração e iteração contínua até a convergência.

 Como testar
O código está pronto para ser executado em qualquer ambiente Python  ou no Google Colab.
